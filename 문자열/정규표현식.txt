정규 표현식

문자열의 패턴은 내용의 앞뒤에 /를 삽입
/패턴/

x -> 문자x
xyz -> 문자열xyz
[xyz] -> x,y,z중 하나의 문자
[a-z] -> a~z중 하나의 문자. [a-f],[A-Za-z]가능
[^xyz] -> x,y,z 외 하나의 문자
[^a-z] -> a~z 외 하나의 문자
abc|xyz -> 문자열 abc 혹은 xyz
{숫자} -> 반복 횟수
^x -> 시작 문자x 
x$ -> 종료 문자x
. -> 하나의 문자(줄바꿈 제외)
x* -> 0개 이상 계속되는 문자
\ -> 다음에 오는 이스케이프 처리
\d -> 숫자[0-9]
\D -> 숫자가 아닌 문자[^0-9]
\w -> 영문,숫자,언더바[A-Za-z0-9_]
\s -> 공백 문자(스페이스, 탭, 줄바꿈)
\S -> 공백 문자 이외의 문자[^\s]
\t -> 수평 탭
\n -> 줄바꿈
+ -> 한개 이상
